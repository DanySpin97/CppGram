Detailed Description

A inline query choosed by the user.

Represents a result of an inline query that was chosen by the user and sent to their chat partner. (https://core.telegram.org/bots/api#choseninlineresult)

Optional. Identifier of the sent inline message.

Available only if there is an inline keyboard attached to the message. Will be also received in callback queries and can be used to edit the message.
